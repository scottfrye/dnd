# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Support for Python 3.12, 3.13, 3.14 and above
- Support for Anaconda/Miniconda environments
- Support for Windows and macOS platforms (in addition to Linux)
- CONTRIBUTING.md with branching and merge strategy guidelines
- CI testing on Windows, macOS, and Linux
- Version tracking with CHANGELOG.md

### Changed
- Minimum Python version updated from 3.9 to 3.12
- Pull requests now target `development` branch by default
- `main` branch reserved for manual releases only

## [0.1.0] - Initial Release

### Added
- Basic project structure
- Core entity system
- Dice rolling mechanics
- Configuration management
- Data validation system
- World state management
- Logging configuration
- Test suite with 165+ tests
